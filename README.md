# Rent Management System API
**A containerized backend for property management and rental tracking.**

## 🚀 Overview
This is a RESTful API designed to manage property rentals, tenant information, and payment records. Built with a focus on **System Reliability** and **Environment Parity**, the project demonstrates a complete backend workflow from database schema design to containerized deployment.

## 🛠 Tech Stack
* **Framework:** [FastAPI](https://fastapi.tiangolo.com/) (Asynchronous Python)
* **Database:** SQL (SQLite for local development / PostgreSQL ready)
* **Containerization:** Docker
* **Environment:** GitHub Codespaces / Linux Containers
* **Documentation:** Swagger UI / ReDoc

## ⚙️ Key Engineering Features
* **Environment Parity:** Fully dockerized to ensure the application runs identically on Windows, Linux, and Cloud environments.
* **Asynchronous Logic:** Utilizes Python's `async/await` for high-performance, non-blocking API endpoints.
* **Relational Schema:** Designed a structured SQL database to maintain data integrity between Tenants, Units, and Payments.
* **Automated Testing & Docs:** Integrated Swagger UI for real-time endpoint verification.
